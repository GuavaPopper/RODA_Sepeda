### **1. UI/UX Design (Rp 2.000.000 - 5.000.000)**

**Apa yang dikerjakan:**
- Research & user flow (3-5 hari)
- Wireframe low-fidelity (2-3 hari)
- Mockup high-fidelity semua halaman (5-7 hari)
- Design system (color, typography, components) (2 hari)
- Prototype interaktif Figma (2-3 hari)
- Revisi (2-3x iterasi)

**Total Estimasi:** 15-20 hari kerja

**Halaman yang di-design:**
- Landing page
- Katalog sepeda (list + detail)
- Booking flow (3-4 halaman)
- User dashboard
- Admin dashboard (5-7 halaman)
- Payment page
- Profile, history, dll

**Rate Profesional:**
- Junior Designer: Rp 100.000 - 150.000/hari ‚Üí 20 hari = Rp 2.000.000 - 3.000.000
- Senior Designer: Rp 200.000 - 300.000/hari ‚Üí 20 hari = Rp 4.000.000 - 6.000.000

**ALTERNATIF MURAH:**
- Mahasiswa/freelancer pemula: Rp 50.000 - 75.000/hari ‚Üí **Rp 1.000.000 - 1.500.000**
- Pakai template UI kit gratis (Tailwind UI, shadcn/ui) ‚Üí **Rp 500.000 - 1.000.000** (custom adjust aja)

### **2. Frontend Development (Rp 5.000.000 - 10.000.000)**

**Apa yang dikerjakan:**
- Setup project (React/Next.js, Tailwind, config) (1-2 hari)
- Convert design ke component (10-15 halaman) (10-15 hari)
- Routing & navigation (1 hari)
- Form handling & validation (3-4 hari)
- State management (Redux/Context) (2-3 hari)
- API integration (fetch data, CRUD) (5-7 hari)
- Responsive mobile & tablet (3-5 hari)
- Testing & bug fixing (5-7 hari)

**Total Estimasi:** 30-40 hari kerja

**Rate Profesional:**
- Junior Frontend Dev: Rp 150.000 - 200.000/hari ‚Üí 35 hari = Rp 5.250.000 - 7.000.000
- Senior Frontend Dev: Rp 250.000 - 400.000/hari ‚Üí 35 hari = Rp 8.750.000 - 14.000.000

**ALTERNATIF MURAH:**
- Mahasiswa IT/bootcamp graduate: Rp 75.000 - 100.000/hari ‚Üí **Rp 2.625.000 - 3.500.000**
- Freelancer remote (Vietnam, Filipina): $10-15/hari ‚Üí **Rp 2.000.000 - 3.000.000**

---

### **3. Backend Development (Rp 7.000.000 - 15.000.000)**

**Apa yang dikerjakan:**
- Database design & schema (2-3 hari)
- Setup server (Node.js/PHP, Express/Laravel) (1-2 hari)
- Authentication & authorization (JWT, role-based) (3-4 hari)
- API endpoints (CRUD bikes, bookings, users, payments) (10-15 hari)
- Business logic (booking validation, availability check, pricing) (5-7 hari)
- Payment gateway integration (Midtrans) (3-5 hari)
- File upload (foto sepeda) (1-2 hari)
- Email notification service (2-3 hari)
- Security implementation (validation, sanitization, rate limiting) (2-3 hari)
- Testing & bug fixing (5-7 hari)
- Deployment & server config (2-3 hari)

**Total Estimasi:** 35-50 hari kerja

**Rate Profesional:**
- Junior Backend Dev: Rp 175.000 - 250.000/hari ‚Üí 42 hari = Rp 7.350.000 - 10.500.000
- Senior Backend Dev: Rp 300.000 - 500.000/hari ‚Üí 42 hari = Rp 12.600.000 - 21.000.000

**ALTERNATIF MURAH:**
- Mahasiswa IT: Rp 100.000 - 125.000/hari ‚Üí **Rp 4.200.000 - 5.250.000**
- Freelancer remote: $12-20/hari ‚Üí **Rp 3.000.000 - 5.000.000**
- Pakai backend as a service (Supabase/Firebase) ‚Üí **Rp 2.000.000 - 3.000.000** (coding lebih cepat)

---

### **4. Testing & QA (Rp 1.500.000 - 3.000.000)**

**Apa yang dikerjakan:**
- Manual testing semua fitur (5-7 hari)
- Bug reporting & retesting (3-5 hari)
- Cross-browser testing (Chrome, Firefox, Safari, Edge) (1-2 hari)
- Mobile responsive testing (Android, iOS) (1-2 hari)
- User acceptance testing (UAT) koordinasi dengan owner (2-3 hari)
- Performance testing (loading speed, stress test) (1-2 hari)

**Total Estimasi:** 13-20 hari kerja

**Rate Profesional:**
- QA Tester: Rp 100.000 - 150.000/hari ‚Üí 15 hari = Rp 1.500.000 - 2.250.000

**ALTERNATIF MURAH:**
- Developer sendiri yang testing (sudah include di development cost) ‚Üí **Rp 0**
- Mahasiswa/intern: Rp 50.000/hari ‚Üí **Rp 750.000 - 1.000.000**
